主要基于yt-dlp/python3/ffmpeg。

普通版带有环境检测与补齐（仅Debian及其发行版），Lite版无此功能。

支持的功能：

============================

cookies.txt / Firefox / Chrome 三种认证方式

单链接下载 / 批量下载模式

视频下载（MP4 最佳画质）

音频提取（MP3 最高音质）

VTT 字幕下载（支持英/日/简体中文及组合）

VTT 转 SRT 智能去重（解决 YouTube 实时字幕累积重复问题）

封面缩略图下载（JPG）

无限重试下载

============================
